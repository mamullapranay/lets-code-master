# Features
- Invite Friends through Friend Requests.
- Create Their Own work Room.
- Invite Peaple to the work Room.
- People can work togeather in the same Room.
- Code to gather on the same file
- And execute the code.

# Build Using
- React Js
- Node Js
- Express Js
- HTML, SCSS, CSS, JS
- Rest API
- MongoDb (Database)
